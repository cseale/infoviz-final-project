# infoviz-final-project
Data Vizualisation 2018 InfoViz Final Project
# Set up
1. install Elasticsearch https://www.elastic.co/downloads/elasticsearch
1. start Elasticsearch
1. `npm run install`
1. `npm run build`
1. `npm run load`
1. `npm run start`
1. Load localhost:80
